# 🏢 Enterprise Product Management System

## 📌 Project Overview

This is a backend application developed using **Spring Boot** to manage product records efficiently.
It provides secure REST APIs with **JWT-based authentication and authorization**.

---

## 🚀 Tech Stack

* Java
* Spring Boot
* Spring Security
* JWT (JSON Web Token)
* Spring Data JPA
* MySQL
* REST APIs
* Postman (for testing)

---

## 🔥 Features

* 🔐 Secure Authentication using JWT
* 📦 Product CRUD Operations (Create, Read, Update, Delete)
* 🏗 Layered Architecture (Controller, Service, Repository)
* 🛢 Database Integration with MySQL
* ⚡ Clean and maintainable code

---

## 📂 Project Structure

src/
├── controller/
├── service/
├── repository/
├── model/
└── config/

---

## 🛠 API Endpoints (Sample)

* POST /auth/login → Login user
* POST /products → Add product
* GET /products → Get all products
* PUT /products/{id} → Update product
* DELETE /products/{id} → Delete product

---

## ▶️ How to Run

1. Clone the repository
2. Open in IDE (Eclipse/IntelliJ)
3. Configure MySQL database
4. Run Spring Boot application
5. Test APIs using Postman
